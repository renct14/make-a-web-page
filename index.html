Lesson 1 - The Basics of the Web and HTML

The Web

The web are numerous computers communicating with each other. Through your browser (e.g. Chrome or Firefox) you can send HTTP requests to servers through the internet. That server will then fetch and return the appropriate HTML documents, which your web browser will interpret and display back to you. The web mainly contains HTML documents, but can also have many other things including images, videos, PDFs, etc.

HTML

HTML stands for Hyper Text Markup Language. HTML is made up of:

  • text content - what you see
  • markup - what it looks like
  • references to other documents - e.g. images and videos
  • links to other pages

A HTML element is made up of an opening tag followed by the content and a closing tag. A void tag is a type of tag that has no content.

Another thing to note is that any whitespace in HTML will automatically turn into a single space.

Tags and Elements

There are many different tags, which define the way the content looks and what it does. Most tags have an opening and a closing tag, which determines where to start and end the function of that tag. All tags begin with a < and end with a >.

Below are some examples of tags are and their functions:

  • <b>bold</b> - makes characters bold
  • <i>italic</i> - makes characters italic
  • <u>underline</u> - underline characters
  • <br> - inserts a break and starts a new line
  • <img> - displays an image from a specified link

Lesson 2 - Creating a Structured Document

Why HTML and CSS?

HTML and CSS are good languages to learn for an introduction to programming. This is because HTML and CSS readily translates codes to something that is visual, making it easy to learn. The logic and processes for writing HTML and CSS will also be useful and applicable to general programming.

Tree-like Structure

HTML documents tend to have nested or tree-like structures. What does this mean? HTML elements can have other elements inside of them. For example, a web page can be viewed as a series of nested boxes, and there can be boxes inside of boxes inside of more boxes.

Lesson 3 - Adding CSS Style and HTML Structure

What is CSS?

CSS stands for Cascading Style Sheets and is a style sheet language that is used to describe the way you want your HTML document to look. With CSS, you can group styles together (e.g. with classes) so you can easily re-apply these formatting. This will largely reduce the amount of code you need to write when creating many similar entities on your website.

Click here to go to Wikipedia for more information on CSS.

Avoiding Repetition

The key advantage of using CSS is to avoid repetition. Why is this so important?

  • To avoid errors: Carrying out repetitive action, possibly with minor changes could often lead to errors, even for the most diligent and detailed programmers. After all, we're only human.
  • To have consistency: Specifying a style through CSS allows for things on your web page to look the same. This will also lead to a more professional finish and look.
  • Ease of making changes: By having all the styles all in one place, it is much easier and significantly faster if edits were to be made to how things looked on your web page. You don't have to look through and manually apply the same style over and over again in the HTML document itself.

These are very useful advantages to using CSS.
